
CodeceptJS tests
------------------


### Requirements
 * `Node` version >= 8.9
 * `Java` verion >= 8

There are positive and negative tests. Also a script to create a new user(create_new_user). They are located in the tests folder. Each check is in a separate file.

How to run tests
--------------------------------------------------------------------------------------------------------

1. To clone the git clone project https://github.com/azarovsergey1988/SSP.git
2. Install npm i codeceptjs -g
3. Go to the project directory and run 'npm i'
4. Run the 'npm run run-ssp' tests
